# Redes Neurais em Python - Livro de Luiz Tiago Wilcke

Este repositório contém 34 scripts fundamentais e avançados de Redes Neurais implementados do zero usando apenas **NumPy**.

## Estrutura do Projeto

### 01. Básico
Scripts iniciais cobrindo modelos lineares, Perceptron simples e fundamentos.
- `01_perceptron_e.py`
- `02_perceptron_ou.py`
- `03_mcculloch_pitts.py`
- ...

### 02. Intermediário
Implementações de multicamadas, otimizadores e técnicas de regularização.
- `11_percepton_multicamadas_xor.py`
- `12_retropropagacao_manual.py`
- ...

### 03. Avançado
Arquiteturas de ponta como CNN, RNN, LSTM, GAN, Transformers e Reinforcement Learning.
- `31_framework_diferenciacao_automatica.py` (Autograd)
- `32_transformer_mestre.py`
- `34_motor_xadrez_deep_learning.py` (Script Principal: +500 linhas)
